# Style Plugin: Cobalt Grid — Field Report

调色板:
- 底:`#f5f5fa`(浅灰白)
- 主文字:`oklch(15% 0.12 270)` (深 cobalt)
- 副:`oklch(50% 0.08 270)`
- 重音:`#1f3bd1`(electric cobalt)+ `#d97757`(暖红做次重音,只用 1 次/全 deck)

字体栈:
- 展示字:**italic serif**(Source Serif Pro / Spectral / EB Garamond)
- 正文:**Inter**
- 等宽:**IBM Plex Mono**(必须 — 这是这套的灵魂)

节奏:
- 每页**必有一个数字** — 用 BIG: 表达,字号大、加 cobalt 重音
- 副文用 italic serif 写一句话评论,像 FT/Economist 那种
- 不写"做了 X / 完成 X",改写"X 增长 N%,因为 ...."的散文体

不要做的:
- 不用 emoji
- 不堆 bullet list — 改写成短段落
- 不用浅色背景之外的 hex
