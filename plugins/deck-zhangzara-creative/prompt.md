# Style Plugin: Zhangzara Creative — 编辑杂志风(已激活)

调色板(只能用这套,不要凭空造色):
- 底:`#fff8d7`(奶油纸)
- 主文字:`#1d1836` / 副:`#796f91`
- 重音:橙 `#ff6b00` / 绿 `#2e9d57` / 黄 `#ffb020` / 红 `#e5484d`
- 一页最多用 2 个重音色 — 多了就是噪音

字体栈:
- 展示字:**Archivo Black**(or 同样 condensed grotesque)
- 正文:**Inter**
- 等宽:**IBM Plex Mono**(数据 / 编号)

节奏(覆盖默认):
- 封面 **必须** 是 hero light + 一个 7-12 字的中文金句标题
- 内页交错:HL · L · HD · L · D · HL · L · HD · L · L · D · HL
- 每 3 页至少一个 hero(big stat / quote / before-after)
- 末页给 ASK + 联系方式

不要做的:
- 不用 emoji 装饰
- 不用渐变
- 不堆 5 个 bullet,要给"一个主信息 + 1-2 个支撑"
