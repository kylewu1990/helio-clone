# Style Plugin: Modern Minimal — Linear/Stripe/Vercel

调色板:
- 底:`#fafafa` 或 `#ffffff`
- 主文字:`oklch(15% 0.02 270)`
- 副:`oklch(60% 0.02 270)`
- 重音:`oklch(60% 0.18 250)`(蓝)— 全 deck 只用 1 个重音色,每页最多 1 处

字体栈:
- 系统字体一以贯之:`-apple-system, BlinkMacSystemFont, 'SF Pro Display', 'Inter'`
- 等宽:`'SF Mono', 'JetBrains Mono'`

节奏:
- 极度留白:每页**只放一个主信息** + 1-2 个支撑
- 标题 30-40px,正文 14-15px,大量行距
- 用 BIG: 表达数字时,数字字号 80+
- 表格/对比用细线分隔,不用颜色填充

不要做的:
- 不用渐变
- 不用色块填充背景
- 不用 emoji
- 不堆 5 个 bullet — 改成"一个事实 + 一个结论"
