# Style Plugin: Scatterbrain — Post-it

调色板:
- 底:`#ebe3d2`(暖米)
- 主文字:`#1c1c1c`
- 副:`#6b5b3e`
- 重音(pastel,可同时用 2-3 个):
  - 黄 `#fcd34d`
  - 粉 `#fda4af`
  - 绿 `#86efac`
  - 蓝 `#93c5fd`

字体栈:
- 展示字:**Permanent Marker / Caveat / Patrick Hand**(手写感)
- 正文:**Inter**
- 等宽:monospace

节奏:
- **每页是一组 sticky notes**,3-6 张便签,每张 1 句话(每条 bullet 就是一张便签)
- 便签颜色错落,不要全一个色
- 标题写得像问题或感叹:"为什么 X?"、"X 不行!"
- 末页:一句话总结 + "下周一件事"

不要做的:
- 不要正经的 KPI 表格
- 不用 BIG: 大数字(太正式)
- 改用 QUOTE: 表达一句话观点
